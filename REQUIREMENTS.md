# 风控数据分箱工具需求文档

## 1. 项目背景
为风控数据分析师提供一个商业化标准、美观、高效的数据分箱工具（桌面版），辅助评分卡模型的变量分析与筛选。工具需具备完整的交互界面，支持从数据导入到分箱调整、结果导出的全流程。

## 2. 用户角色
- **风控数据分析师**：负责变量分析、WOE编码、模型变量筛选。
- **模型开发人员**：使用分箱结果进行逻辑回归等评分卡模型开发。

## 3. 功能需求

### 3.1 数据接入与预处理
- **数据导入**：支持 CSV、Excel 等常见格式。
- **数据持久化与项目管理**：
    - **项目初始化**：导入数据时自动创建项目文件夹结构。
    - **原始数据存储**：将导入的原始数据备份存储（如 parquet/csv）。
    - **过程快照 (Snapshot)**：
        - 支持用户手动点击“保存”生成快照。
        - 快照内容包括：当前分箱配置（切点、算法参数）、变量EDA分析结果、分箱结果表。
        - 支持从历史快照**重新加载**，恢复到之前的分析状态。
    - **退出保护**：关闭软件时检测是否有未保存的更改，并提示用户保存。
- **数据查看与预览**：
    - 提供数据预览网格 (Data Grid)。
    - **大数据支持**：对于百万级行数，支持**分页加载**或**虚拟滚动 (Virtual Scrolling)**，避免前端卡顿。
    - **交互操作**：支持对数据进行**列筛选 (Filter)**、**排序 (Sort)** 和 **搜索 (Search)**。
- **变量选择与探索性分析 (EDA)**：
    - 自动识别数值型与类别型变量，允许用户选择目标变量（Target/Label）和自变量（Features）。
    - **智能分箱建议**：根据变量特征（如分布形态、唯一值数量、与Target的相关性）推荐最合适的分箱算法。
        - 例如：对于长尾分布建议卡方分箱；对于高基数类别变量建议决策树分箱。
    - **自变量详情概览**（选中变量时展示）：
        - **基础统计**：类型、样本数、空值数、空值占比、唯一值数量 (Unique Count)。
        - **数值型分布**：最小值、最大值、均值、中位数、标准差、分位数 (25%, 50%, 75%)、偏度、峰度。
        - **可视化分布**：
            - **数值型**：直方图 (Histogram) 或 核密度估计图 (KDE)，展示数据分布形态（如正态、长尾）。
            - **类别型**：条形图 (Bar Chart)，展示各类别频数。
        - **异常值检测**：简单的箱线图 (Box Plot) 或 Z-score 提示。
        - **目标相关性**：展示该变量与 Target 的初步关系（如相关系数或分组 Bad Rate）。

### 3.2 核心分箱功能
支持多种自动与半自动分箱算法：
1.  **等频分箱 (Equal Frequency)**
2.  **等距分箱 (Equal Width)**
3.  **卡方分箱 (ChiMerge)**
4.  **决策树分箱 (Decision Tree)**
5.  **Best-KS 分箱**（最大化 KS，支持箱数与最小样本约束）
6.  **自定义切点**：用户可手动输入或调整切分点。

### 3.3 约束与调整 (高级交互)
- **单调性约束**：支持设置 Bad Rate 严格单调递增或递减。
- **缺失值处理**：
    - **忽略**：不参与分箱计算。
    - **单独成箱**：缺失值作为独立的一箱。
    - **智能归并**：归并到 Bad Rate 最接近的箱或特定的目标箱。
- **交互式调整**：
    - **可视化微调**：在直方图/分布图上通过**游标 (Cursor)** 或 **拖拽 (Drag & Drop)** 直接调整切分点。
    - **切点操作**：支持**增加**、**删除**切点，或**手动输入数值**精确修改切点。
    - **视图缩放**：支持坐标轴的**缩放 (Zoom)** 和 **平移 (Pan)**（利用鼠标滚轮和右键），方便查看局部数据密度。
    - **实时重算**：调整切点后，点击“确认”或自动触发分箱统计指标（WOE, IV, Bad Rate）的重新计算与刷新。

### 3.4 结果分析与可视化
- **分箱明细表**：包含以下指标：
    - 箱范围 (Bin Range)
    - 样本数 (Count) & 占比 (Percent)
    - 坏样本数 (Bad Count) & 坏样本率 (Bad Rate)
    - 好样本数 (Good Count) & 好样本率 (Good Rate)
    - WOE (Weight of Evidence)
    - IV (Information Value)
    - Lift (提升度)
- **可视化图表 (美观与交互)**：
    - **Bad Rate 趋势图**：展示各箱 Bad Rate 的变化趋势。
    - **分箱分布图**：展示各箱样本占比。
    - **交互增强**：
        - **悬停提示 (Hover Tooltip)**：鼠标移动到数据点或柱子上时，实时显示详细信息（如：箱范围、样本数、占比、Bad Rate、WOE等）。
        - **图表联动**：选择分箱图中的某一箱时，同步高亮明细表中对应的行（可选）。
        - **美观性**：使用现代配色方案，图表清晰整洁，符合商业化标准。

### 3.5 导出
- 导出分箱明细表（Excel/CSV）。
- 导出分箱规则代码（Python/SQL），可直接用于生产环境。

## 4. 非功能需求
- **用户界面 (GUI)**：
    - **框架**：**PyQt6 / PySide6** (原生桌面应用)。
    - **风格**：美观、现代、简洁（需使用 QSS 或现成的 UI 库如 `qt-material` 进行美化）。
    - **性能**：响应速度快，操作流畅。
    - **标准**：达到商业化软件交付标准。
- **性能与并发**：
    - **百万级支持**：核心计算逻辑需优化，支持百万级数据量在合理时间内完成。
    - **异步处理**：对于耗时较长的计算（如大数据量导入、自动分箱计算），必须使用后台线程 (`QThread`) 处理，避免阻塞主 UI 线程。
    - **进度反馈**：前端界面必须提供明确的进度条 (`QProgressBar`) 或加载状态提示。
- **易用性**：操作流程符合风控建模直觉。

## 5. 测试与验收
- **测试数据**：需生成一份包含 20000 行、30 个自变量、1 个目标变量的 Excel 数据集。自变量需与目标变量具有一定的相关性，用于验证分箱效果。
- **自动化测试**：编写 Python 脚本，自动化测试核心分箱算法的准确性、稳定性和异常处理（如缺失值）。

## 6. 技术栈 (更新)
- **语言**：Python 3.9+
- **GUI 框架**：**PyQt6** (Core)
- **图表库**：**PyQtGraph** (高性能交互) 或 **Matplotlib** (嵌入式，绘图质量高)
- **数据处理**：pandas, numpy, openpyxl (Excel读写)
- **算法支持**：scikit-learn (决策树), scipy (卡方)
 - **打包**：PyInstaller（桌面应用交付）
