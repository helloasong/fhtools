## 2. 任务详细规划

### Task 1: 高级约束参数面板
**目标**: 创建可折叠的高级参数面板
**文件**: `src/ui/widgets/advanced_params_panel.py`

**参数列表**:
| 参数 | 控件类型 | 默认值 | 范围 |
|------|----------|--------|------|
| max_n_prebins | QSpinBox | 20 | 5-200 |
| min_prebin_size | QDoubleSpinBox | 0.05 | 0.01-0.5 |
| min_bin_size | QDoubleSpinBox | None | 0-0.5 |
| max_bin_size | QDoubleSpinBox | None | 0-1.0 |
| min_bin_n_event | QSpinBox | None | 0-1000 |
| min_bin_n_nonevent | QSpinBox | None | 0-1000 |
| max_pvalue | QDoubleSpinBox | None | 0-1.0 |
| gamma | QDoubleSpinBox | 0 | 0-1.0 |

**界面**:
```
┌─────────────────────────────────────────┐
│ [展开 ▼] 高级选项                        │
├─────────────────────────────────────────┤
│ 预分箱数:        [20]    [?]            │
│ 预分箱最小占比:  [0.05]  [?]            │
│ 每箱最小占比:    [0.05]  [?]            │
│ 最大p-value:     [None]  [?]            │
│ 正则化Gamma:     [0]     [?]            │
└─────────────────────────────────────────┘
```

---

### Task 2: 富文本参数提示
**目标**: 创建带图标的富文本提示组件
**文件**: `src/ui/widgets/rich_tooltip_label.py`

**特性**:
- 显示 [?] 图标
- 鼠标悬停显示富文本提示框
- 支持 HTML 格式 (标题、列表、警告)
- 延迟显示 (500ms)

**提示内容示例**:
```html
<h4>📘 最小箱大小 (min_bin_size)</h4>
<p>每箱最小样本占比，防止过拟合。</p>
<ul>
    <li>大数据(&gt;10万): 0.02 (2%)</li>
    <li>中小数据: 0.05 (5%)</li>
</ul>
<p style='color:#e74c3c;'>⚠️ 设置过大可能导致无解</p>
```

---

### Task 3: 求解状态显示
**目标**: 创建求解状态指示器
**文件**: `src/ui/widgets/solve_status_widget.py`

**状态**:
- 🟡 `solving`: "求解中..." (黄色)
- 🟢 `optimal`: "最优解" (绿色)
- 🔵 `feasible`: "可行解" (蓝色)
- 🔴 `infeasible`: "无解" (红色)
- ⚫ `timeout`: "超时" (灰色)

**信息面板**:
- 求解时间
- 目标函数值 (IV)
- 迭代次数
- 约束违反情况
