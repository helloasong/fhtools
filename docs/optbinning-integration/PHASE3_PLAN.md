## 2. 任务详细规划

### Task 1: 分类型变量支持
**目标**: 支持分类型变量分箱 (dtype='categorical')
**文件**: 
- 修改 `optbinning_adapter.py`
- 修改 `optbinning_config_panel.py`

**功能**:
1. 自动检测变量类型 (数值/分类)
2. 分类型变量显示 `cat_cutoff` 参数 (低频合并阈值)
3. 适配器支持 `dtype='categorical'`

**界面变化**:
- 变量列表显示类型图标 (🔢数值 / 🏷️分类)
- 选中分类变量时，参数面板自动调整

---

### Task 2: 批量处理
**目标**: 支持多变量一键分箱
**文件**: 
- 修改 `combined_view.py`
- 修改 `project_controller.py`

**功能**:
1. 多选变量列表 (Ctrl/Shift 多选)
2. "批量分箱"按钮
3. 后台线程执行，进度条显示
4. 结果汇总展示

**界面**:
```
┌─────────────────────────────────────────┐
│ [批量分箱] 已选择 5 个变量               │
├─────────────────────────────────────────┤
│ 🟢 var1 完成  IV: 0.45                  │
│ 🟡 var2 求解中...                       │
│ ⏳ var3 等待中...                       │
│ ...                                     │
└─────────────────────────────────────────┘
```

---

### Task 3: JSON 配置导出
**目标**: 导出 Optbinning 配置为 JSON
**文件**: 
- 新增 `export_optbinning_config.py`
- 修改 `export_view.py`

**功能**:
1. 导出单个变量配置
2. 导出所有变量配置
3. 格式兼容 optbinning 原生格式

**JSON 结构**:
```json
{
  "version": "1.0",
  "export_time": "2026-03-11T10:00:00",
  "project": "credit_score_v1",
  "variables": {
    "age": {
      "dtype": "numerical",
      "splits": [25.0, 35.0, 45.0, 55.0],
      "woe": [-0.5, 0.2, 0.8, 1.2],
      "iv": 0.45,
      "config": {
        "solver": "cp",
        "divergence": "iv",
        "max_n_bins": 5
      }
    }
  }
}
```
