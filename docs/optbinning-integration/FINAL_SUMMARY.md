## 1. 项目概述

成功将 [optbinning](https://gnpalencia.org/optbinning/) 集成到 FHBinningTool，作为高级分箱引擎，提供更专业、更灵活的风控数据分箱能力。

---

## 2. 核心成果

### 2.1 功能矩阵

| 功能类别 | 具体功能 | 状态 |
|----------|----------|------|
| **基础功能** | 最优分箱设为默认 | ✅ |
| | 原有6种算法保留 | ✅ |
| | 动态面板切换 | ✅ |
| | 一键恢复推荐值 | ✅ |
| **高级功能** | 8个约束参数面板 | ✅ |
| | 富文本参数提示 | ✅ |
| | 6种求解状态显示 | ✅ |
| | 单调性趋势指示器 | ✅ |
| **增强功能** | 分类型变量支持 | ✅ |
| | 批量处理 | ✅ |
| | JSON配置导出 | ✅ |

### 2.2 技术栈

- **核心引擎**: optbinning (CP/MIP/LS 求解器)
- **优化目标**: IV / JS / Hellinger / Triangular
- **约束支持**: 单调性、箱大小、p-value、正则化
- **变量类型**: 数值型 + 分类型

## 9. 快速开始

### 9.1 安装

```bash
# 基础依赖
pip install pandas numpy PyQt6

# Optbinning 引擎
pip install optbinning>=0.19.0
```

### 9.2 运行

```bash
python -m src.ui.main_window
```

### 9.3 基本流程

1. 导入数据 (CSV/Excel)
2. 设置目标变量 (右键列名)
3. 选择特征变量
4. 点击 "⚡恢复推荐" (或手动调参)
5. 点击 "运行"
6. 查看分箱结果
