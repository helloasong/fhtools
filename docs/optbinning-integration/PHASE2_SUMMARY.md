## 2. 功能详情

### 2.1 高级约束参数面板 (AdvancedParamsPanel)

**8 个参数配置**:

| 参数 | 类型 | 说明 |
|------|------|------|
| max_n_prebins | int | 预分箱数 (5-200) |
| min_prebin_size | float | 预分箱最小占比 (0.01-0.5) |
| min_bin_size | float? | 每箱最小占比 (0-0.5) |
| max_bin_size | float? | 每箱最大占比 (0-1.0) |
| min_bin_n_event | int? | 每箱最小坏样本数 |
| min_bin_n_nonevent | int? | 每箱最小好样本数 |
| max_pvalue | float? | 最大 p-value |
| gamma | float | 正则化系数 (0-1.0) |

**特性**:
- ✅ 可折叠面板 (200ms 动画)
- ✅ None 值处理 (复选框启用/禁用)
- ✅ 参数变化信号通知

### 2.2 富文本参数提示 (RichTooltipLabel)

**组件**:
- `RichTooltipLabel` - 带图标的富文本提示标签
- `RichTooltipHelper` - 仅 [?] 图标
- `RichTooltipWrapper` - 包装器自动组合

**特性**:
- ✅ 延迟 500ms 显示
- ✅ 支持 HTML 格式
- ✅ 预定义 11 个参数模板

**提示内容示例**:
```html
<h4>📘 最小箱大小 (min_bin_size)</h4>
<p>每箱最小样本占比，防止过拟合。</p>
<ul>
    <li>大数据(&gt;10万): 0.02 (2%)</li>
    <li>中小数据: 0.05 (5%)</li>
</ul>
<p style='color:#e74c3c;'>⚠️ 设置过大可能导致无解</p>
```

### 2.3 求解状态显示 (SolveStatusWidget)

**6 种状态**:

| 状态 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| solving | 🟡 | #f39c12 | 求解中... |
| optimal | 🟢 | #27ae60 | 最优解 |
| feasible | 🔵 | #3498db | 可行解 |
| infeasible | 🔴 | #e74c3c | 无解 |
| timeout | ⚫ | #95a5a6 | 超时 |
| unknown | ⚪ | #bdc3c7 | 未知 |

**信息面板字段**:
- 求解时间
- 目标函数值 (IV)
- 迭代次数
- 约束违反数

### 2.4 单调性趋势指示器

**位置**: 分箱图右上角

**趋势图标映射**:

| 趋势 | 图标 | 标签 |
|------|------|------|
| auto | 📊 | 自动 |
| ascending | ↗️ | 递增 |
| descending | ↘️ | 递减 |
| peak | ⛰️ | 单峰 |
| valley | 🏞️ | 单谷 |

**样式**:
- 白色半透明背景
- 圆角边框
- 小号字体 (11px)

## 4. 使用示例

### 设置求解状态

```python
# 求解中
config_panel.set_solving_status(True)

# 求解完成
config_panel.set_solve_result('optimal', {
    'solve_time': 2.34,
    'objective_value': 0.847,
    'n_iterations': 156,
    'n_constraints_violated': 0
})
```

### 获取完整配置

```python
config = config_panel.get_config()
# 返回包含基础参数 + 高级参数的完整字典
```
