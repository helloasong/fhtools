## 功能补全
- Best-KS 分箱：实现以最大化 KS 为目标的自动分箱（支持箱数上限、最小样本约束），并加入到算法下拉框与控制器。
- SQL 规则导出：为每变量生成 CASE WHEN 片段，保证与切点及缺失策略一致。
- 缺失策略 UI：在 BinningView 加入缺失值策略选择（忽略/单独成箱/归并），实时重算与持久化到快照/导出。
- 快照加载入口：在 Import/Export 页新增“加载快照”按钮，恢复之前的分析状态。

## 交互与性能
- 双轴图完善：左轴柱状（样本数），右轴折线（Bad Rate），统一调色与图例；优化悬停 Tooltip（显示箱范围、样本数、Bad Rate、WOE、IV）。
- 虚拟滚动表：用 `QAbstractTableModel` + 分页/窗口化加载实现百万级数据预览，支持列筛选、排序、搜索。
- 异步与进度：导入、分箱、导出、快照均使用 `QThread`，在 UI 显示不定/分阶段进度，避免界面阻塞。

## 导出与配置
- Excel 报告美化：数值格式统一、列顺序固定、自动列宽、对 Summary 添加总 IV 与特征排序。
- Python 规则扩展：在 `transform(df)` 中支持缺失策略与可选 WOEs 输出开关；生成注释说明。
- 配置文件：新增 `config.json`（主题、默认初始箱数、导出目录、并发参数）并在启动时加载。

## 测试与验收
- 单元测试：Best-KS、SQL 导出、缺失策略持久化与导出一致性、双轴图数据管线。
- 端到端测试：导入→EDA→Best-KS→微调→导出（Excel/Python/SQL）→快照保存/加载；记录耗时并设定基线。
